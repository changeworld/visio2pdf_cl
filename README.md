# Visio2PDF

[![MIT License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

Visio to PDF format.

## Usage

1. git clone https://github.com/changeworld/visio2pdf_cl.git
2. cd visio2pdf_cl
3. ruby visio2pdf.rb [Existing directory of the visio file] .
